# Maven Tips and Tricks

## run single test

```bash
mvn -Dtest=ProsiVersiMapperTest test
```

## create dependency tree

```bash
mvn dependency:tree > tree.txt
```

## Environment variables for SpringBoot maven tests

Configure them only in the maven-surefire-plugin.
They will be available to:
- all @SpringBootTest
- all unit tests
- all integration tests executed by Surefire

````xml
<plugin>
  <artifactId>maven-surefire-plugin</artifactId>
  <configuration>
    <environmentVariables>
      <ECOHUB_MINIO_ENDPOINT>
        http://localhost:9000/test-bucket
      </ECOHUB_MINIO_ENDPOINT>
    </environmentVariables>
  </configuration>
</plugin>
````
