# Git Tips and Tricks

##  check if file is excluded by .gitignore

```bash
git check-ignore -v filename
```

##  revert last commit

```bash
git reset --hard HEAD~1
```

##  change last commit message

```bash
git commit --amend <REF>
  => enter new commit message
```

## .gitignore a file after it was commited

```bash
- add to .gitignore
- git rm -r --cached . (removes all files ?)
- git add -A (add all files again)
```

## amend commit without new message

```bash
git commit --amend --no-edit
(--no-edit means: do not change commit message)
```
